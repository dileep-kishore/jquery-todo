# todolist

An application made using nodejs, express, mongodb and jQuery.

## Install

1. Install [mongodb](https://www.mongodb.com/download-center?jmp=nav#community)
2. Install node packages
```sh
npm install
```
3. Install nodemon [Optional]


## Run

```sh
node index.js
# or
nodemon index.js
```
